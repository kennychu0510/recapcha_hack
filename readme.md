# Cracking Recapcha with HTML Canvas and Tesseract.js

**Techniques Used:**
1. Render image with HTML Canvas.
2. Convert image into black and white.
3. Removing noise by manipulating neighbor pixels.
2. Run OCR on cleaned image.
